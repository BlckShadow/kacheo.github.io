Linux Kernel Rootkit
A linux kernel module which masks the trace of the kernel and its activities. It allows one to ssh into the "hacked" machine with sudo access. By manipulating function pointers and modifying the output of the functions, one can change the output of some commands on terminal.

Animated Sprite Editor
This Java Swing Application lets users create sprites and save/load these sprites. You can even animate the sprites within the application. The user can speed up/slow down the animations. Within the application, it features a drawing palette that lets users draw out sprite frames using ellipses, squares, and lines. The application uses XML for file I/O.
Social Networking Site
This web application which uses JSP/Servlets to create a social networking experience similar to Google+ and Facebook. Users can create circles, add their friends on it and like the posts. The website also features functionality for Managers and Employees whom can add advertisements to the database. The user can then search for advertisements and purchase items.
